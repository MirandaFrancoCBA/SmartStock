# SmartStock

Inventory management system built with:

- Django REST Framework
- Angular
- PostgreSQL
- Docker

Project created for portfolio and learning purposes.

| Librería      | Para qué             |
| ------------- | -------------------- |
| Django        | framework            |
| DRF           | API REST             |
| psycopg2      | PostgreSQL           |
| dotenv        | variables de entorno |
| django-filter | filtros API          |
| drf-yasg      | Swagger docs         |
